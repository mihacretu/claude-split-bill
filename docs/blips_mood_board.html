Design Philosophy

The Split Bill app embraces a clean, modern aesthetic with glass morphism elements and soft gradients. The design prioritizes usability while maintaining visual elegance through subtle shadows, thoughtful spacing, and a harmonious color palette that guides user interaction naturally.
